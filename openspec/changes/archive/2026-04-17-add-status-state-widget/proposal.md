## Why

В интерфейсе нет явного способа показать пользователю текущее состояние операций и данных. Из-за этого невозможно единообразно отобразить сценарии, когда ничего не происходит, идет загрузка или возникла ошибка.

## What Changes

- Добавить переиспользуемый виджет состояния для GTK4-приложения.
- Поддержать три отображаемых состояния: без активного процесса, загрузка, ошибка.
- Встроить виджет в основное окно как пример использования и переключать его из существующих обработчиков событий.
- Задать понятные текстовые сообщения и визуальные индикаторы для каждого состояния.

## Capabilities

### New Capabilities
- `status-state-widget`: Переиспользуемый UI-компонент, который отображает idle-, loading- и error-состояния приложения.

### Modified Capabilities
- None.

## Impact

- Затронуты `src/main.vala` и, при необходимости, `src/meson.build`.
- Новая логика состояния появится в GTK4 UI-слое приложения.
- Внешние зависимости и публичные API проекта не меняются.
